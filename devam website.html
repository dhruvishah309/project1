Automated Financial Modelling Report

Over time, I developed a deep interest in financial markets, and it has become something I study and learn about every single day. To sharpen my skills, I built a complete automated system that could generate a professional-grade report by simply just downloading the company from Screener. The report included:

  • A One-Page Company Profile
  • Automated Historical Financial Statements
  • Common Size Statements & Ratio Analysis
  • DuPont Breakdown with insightful summary
  • Forecasting Revenue, EBITDA & EPS using regression
  • Beta calculation using market regression
  • WACC, Intrinsic Growth, & Reinvestment Rate computations
  • A Full DCF Valuation Model
  • Comparable Company Valuation
  • Football Field Analysis
  • Altman’s Z-Score Analysis
  • Value at Risk (VaR) using both Historical Approach & Monte Carlo Simulation

What made this unique was that I automated the process to ensure clean, structured, and insightful outputs in one go - a tool that could essentially be applied to any listed company.

Bank Performance Analysis Dashboard (Excel & VBA Project)

Alongside financial modeling, I have developed strong expertise in Microsoft Excel and VBA Macros. One of my most impactful projects was a Bank Performance Analysis for FY 2024. This dashboard ranked banks using multi-criteria decision-making (MCDM) methods and gave users the flexibility to select any number of banks for comparison.

  • The weights were derived using the Fuzzy AHP method, though users also had the option to dynamically adjust the weights based on their own analysis.
  • The final rankings were computed using Fuzzy TOPSIS, applied either on the Fuzzy AHP weights or the user-defined dynamic weights.
  • To make the dashboard highly interactive, I integrated a VBA-powered chatbot assistant with 30 in-built queries. Users could select questions, and the assistant would automatically pull data, generate insights, and even plot relevant graphs on demand.

This project combined advanced decision-making techniques with interactivity, creating a tool that was not only analytical but also user-friendly and dynamic.
